Imao Jokes API

The Imao API uses a RESTful JSON interface. Requests to the API are not rate limited and free of charge and current features of api will always remain free of charge even if we added premium features in future.


Get Jokes

Currently there are about 15,000 handpicked jokes from across internet and we are adding more very soon.

  • To get one random joke
    • URL : http://imao.herokuapp.com/api/jokes/random
    Method : GET
  • To get 'n' random jokes

    URL : http://imao.herokuapp.com/api/jokes/n
    Method : GET

    • e.g.http://imao.herokuapp.com/api/jokes/5
    (to get 5 random jokes)
